## Project Description:-
A lightweight browser extension that tracks the amount of time a user spends on different websites. It helps users understand browsing habits and improve productivity by monitoring time usage in real time.

## Features:-
- Tracks time spent on each website
- Automatically detects active tabs
- Stores browsing time locally
- Simple and user friendly interface
- Runs in the background

## Technologies Used:-
- HTML
- CSS
- JavaScript
- Chrome Extensions API
- Manifest V3

## How to Test:-
1.Open multiple websites in different tabs
2.Switch between tabs
3.Open the extension popup
4.View the time spent on each website

## How it runs:
Browser loads: manifest.json

Browser runs:
background.js (service worker)
script.js (popup/content)
